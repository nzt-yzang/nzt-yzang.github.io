    可以先求出每个数的因数出现的个数,,然后发现k个数均含有某个因数(即某个因数出现的次数为k),那么这个数必为这k个数的公因数,其中找出最大的即可,

#include <bits/stdc++.h>
using namespace std;
int n,t,c[1000001];
int main()
{
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		int a;
		cin>>a;
		int m = sqrt(a);
		t = max(t,a);
		for(int i = 1 ; i <= m ; i++)
		{
			if(a%i == 0)
			{
				c[i]++;
				if(i*i != a) 
				c[a/i]++;
			}
		}
	}
	int x = t;
	for(int i = 1 ; i <= n ; i++)
	{
		while(c[x] < i)x--;
		cout<<x<<endl;
	}
	return 0;
}
  • P2651 添加括号III
    a1肯定是分子，a2肯定是分母，所以让a3~an的为分子就行,然后一边输入一边用a[2]约分,最后判断a[2]是否为1就行
#include <bits/stdc++.h>
using namespace std;
int a[100001];
int gcd(int a,int b)
{
	if(b == 0) return a;
	return gcd(b,a%b);
}
int n,t;
int main()
{
	cin>>n;
	for(int i = 1 ; i <= n ; i++)
	{
		cin>>t>>a[1]>>a[2];
		a[2] /= gcd(a[1],a[2]);
		for(int i = 3 ; i <= t ; i++)
		{
			cin>>a[i];
			a[2]/=gcd(a[2],a[i]);
		}
		if(a[2] == 1)
		cout<<"Yes"<<endl;
		else
		cout<<"No"<<endl;
	}
	return 0;
}

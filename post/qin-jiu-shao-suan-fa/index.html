秦九韶算法

2020-03-18
  • P2312 解方程

已知多项式方程：a0+a1x+a2x2+..+anxn=0，求方程在[1, m ] 内的整数解（n 和m 均为正整数）
//a0+a1x+a2x2+...+anxn=a0+x(a1+x(a2+x(a3+......anx;

主代码

bool calc(ll x)
{
     sum=0;//一定要清零
     for(ll i=n;i>=1;i--)
  {
         sum=((A[i]+sum)*x)%p;
          //这里套用秦九韶算法求多项式的值 注意A[0]并不需要乘x！
 }
     sum=(sum+A[0])%p;//这里再算上A[0]
     return !sum;//如果答案是0说明x值为该多项式的解，返回1(true) 
}

问题:n个数依次进栈，可随机出栈。求有几种进出栈的可能。

  • P1044 栈
    基于这道题理解公式一
    设f[i]表示i个数的可能,则f[0]=f[1]=1;
    设k为出栈序列的最后一个数,则k有n种取值,由于k是最后一个出栈的,所以可以把已经出栈的数分为两部分
    1,比k大 ,2,比k小
    比k大的数有n-k个,则所有出栈的可能有f[n-k]个
    比k小的有k-1个,,则所有出栈的可能有f[K-1]个
    综上,一个k的取值,所有的可能为f[n−k]∗f[k−1]f[n-k]*f[k-1]f[n−k]∗f[k−1];
    有因为k有n个取值
    所以ans=f[0]∗f[n−1]+f[1]∗f[n−2]+...+f[n−1]∗f[0];ans = f[0]*f[n-1] + f[1]*f[n-2] + ... + f[n-1]*f[0];ans=f[0]∗f[n−1]+f[1]∗f[n−2]+...+f[n−1]∗f[0];
#include <bits/stdc++.h>
using namespace std;
int n,f[2001];
int main()
{
	cin>>n;
	f[0]=f[1]=1;
	for(int i = 2 ; i <= n ; i++){
		for(int j = 0 ; j < i ; j++){
			f[i] += f[j]*f[i-j-1];
		}
	}
	cout<<f[n];
	return 0;
}

基于公式四解这道题
与组合数公式1不同这个是两个组合数的减法
组合数就是一个杨辉三角，可以递推得到,但我们发现对于大数据你要取模，而对于除法你是没办法用膜的性质的(当然你可以应用逆元).而减法是可以用膜的性质的

#include<cstdio>
using namespace std;
int n;
int c[41][21];
int main(){

    scanf("%d",&n);
    for(int i=1;i<=2*n;i++) c[i][1]=c[i][i]=1;
    for(int i=3;i<=2*n;i++)
     for(int j=2;j<i;j++)
      c[i][j]=c[i-1][j]+c[i-1][j-1];
    printf("%d",c[2*n][n]-c[2*n][n-1]);
    return 0;
}

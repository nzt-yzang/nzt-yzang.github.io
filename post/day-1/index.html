1.组合数问题

(高中数学.....我还是太菜了)
计算组合数的一般公式：Cnm=n!m!(n−m)!C_n^m=\frac{n!}{m!(n-m)!}Cnm​=m!(n−m)!n!​
代码:

inline LL ck(LL x)//开long long
{
  if(x==0)return 1;
  int sum=1;
  for(int i=1;i<=x;i++)sum*=i;
  return sum;
}
inline LL C(LL n,LL m)
{
  return ck(n)/(ck(m)*ck(n-m));//组合数公式
}
